# Sun Chaser Brand Guide

## Sun Chaser Palette

### Primary (Brand Yellow)
- Primary: #FFD650 – Main accent for buttons, highlights, chart lines
- Hover: #FFC833 – Slightly deeper for hover or focus
- Active: #E6B300 – Pressed / selected state

### Accent (Brown Family)
- Primary: #99472D – Icons, headers, outlines
- Hover: #B65B3A – Hovered icons or active nav links
- Active: #7F3A22 – Selected or pressed state

### Text / Ink
- Primary Text: #2B211D – Main readable text
- Secondary Text: #5B504B – Labels, helper text
- Inverse Text: #FFFFFF – On yellow or dark backgrounds

### Surface / Background
- Background Base: #FCFCFB – Page background
- Card Surface: #F8F5F2 – Panels, cards, modals
- Surface Alt: #DAD3CC – Table headers, contrast blocks
- Border / Divider: #E6E2DD – Thin lines, grid borders
- Shadow / Depth: rgba(43, 33, 29, 0.12) – Box shadows

### Status / Utility
- Info: #586E75 – Subtle data accents, inactive states
- Warning: #FF9E3D – Alerts or attention elements
- Success: #C7E57F – Positive feedback
- Danger: #E26D5C – Errors or destructive actions


## Typography
- **Font**: CooperHewitt-Semibold
- **Font Author**: Chester Jenkins

## Logo Assets
- **Full Logo**: Available in multiple formats (PNG, SVG)
- **Van Icon**: Isolated van icon for favicons and small UI elements
- **Monochrome**: Black and white versions available

## Usage Notes
- Use yellow background for primary brand elements
- Brown for van icon and accent details
- White text for contrast against yellow background
- Van icon works well as favicon or small UI element
